☕ Coffee Machine Program (OOP Version)
--------------------------------------------
📌 Overview
--------------
Welcome to the Coffee Machine Program! This Python application simulates a coffee vending machine using Object-Oriented Programming (OOP) concepts. It allows users to order coffee, process payments, manage resources, and access an admin mode for maintenance.

🛠️ OOP Implementation
--------------------------
This version utilizes Object-Oriented Programming (OOP) with classes:
CoffeeMachine – Handles operations and resource management.
Menu – Stores coffee options and ingredients.
MoneyMachine – Manages transactions and calculations.
