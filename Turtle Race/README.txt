🐢 Turtle Race Game using Python Turtle
-------------------------------------------------
A fun and colorful racing game built with Python’s turtle module!
Place your bet on a turtle color and watch them race across the screen — will your turtle win? 🏁

🎮 Features
---------------
Lets users bet on a turtle’s color before the race begins
Uses random movement to make every race unique
Displays the winning turtle at the end of each race
Built entirely with Python’s turtle graphics

🧠 Tech Stack
------------------
Python
Turtle Graphics
Random module

💡 How It Works
--------------------
The user places a color bet.
Six turtles line up at the start line.
Each turtle moves forward by random distances.

The first turtle to reach the finish line wins — and you find out if your bet was right!
