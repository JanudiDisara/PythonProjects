🚀 Overview
--------------
Guess the Number is a fun and interactive Python game where you try to guess a randomly selected number between 1 and 100. Choose your difficulty and start guessing!

🎮 How to Play
-----------------
Run the Python script.
The program will randomly pick a number between 1 and 100.

Choose a difficulty level:
🟢 Easy: 10 chances to guess the number.
🔴 Hard: 5 chances to guess the number.

Enter your guess and receive feedback:
📉 "Too high!" if your guess is above the number.
📈 "Too low!" if your guess is below the number.

Keep guessing until:
🎉 You guess correctly (you win!).
❌ You run out of chances (game over!).

✨ Features
-------------
✅ Random number generation
✅ Two difficulty levels
✅ User-friendly prompts and feedback
✅ Keeps track of remaining attempts
