🏓 Pong Game (Python Turtle)
-------------------------------------
A classic Pong arcade game built using Python’s Turtle Graphics module.
This project recreates the two-player Pong experience with smooth paddle movement, ball collision detection, and a dynamic scoreboard.

✨ Features
---------------
Two-player controls (W/S for left paddle, ↑/↓ for right paddle)
Ball movement with realistic bounce mechanics
Collision detection with walls and paddles
Automatic scoring and ball reset after a miss
Simple yet interactive game interface using the Turtle library

🧠 Tech Stack
-----------------
Python
Turtle Graphics

🎮 How to Play
-------------------
Run main.py
Use the controls to hit the ball back and forth.
First to score wins!
