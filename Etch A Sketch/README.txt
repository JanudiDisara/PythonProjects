🖍️ Etch-A-Sketch using Python Turtle
---------------------------------------------
A fun interactive drawing program built with Python’s turtle module — inspired by the classic Etch-A-Sketch toy!
Use your keyboard to move the turtle and draw shapes freely on the screen.

🎮 Controls
---------------
W → Move Forward
S → Move Backward
A → Rotate Left
D → Rotate Right
C → Clear the screen and reset position

🧠 Tech Stack
-----------------
Python
Turtle Graphics

💡 How It Works
--------------------
The program listens for keyboard inputs.
Each key controls the turtle’s movement or resets the drawing.
You can sketch creative patterns directly on the screen.
